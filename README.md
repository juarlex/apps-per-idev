apps-per-idev
==============

This program shows all the installed applications per iDevice connected 
to your computer.

Author
======

* Alejandro Juarez, twitter: @karmatr0n

Requirements
============

* libimobiledevice 1.1.5 or higher
* ruby 1.9.x or highger
* idevice gem

Installation
============

$ git clone 

$ cd apps-per-idev

$ bundle install

Usage
=====

$ ./apps-per-idev
